# tabby-status 项目说明

## 项目路径

`/Users/endlessjy/Documents/AIGC/tabby-status`

## 项目定位

这是一个 Tabby 终端插件，用于在 SSH 连接建立后，在窗口左侧展示远程主机系统状态。面板提供常用监控信息，包括 CPU、内存、交换分区、负载、网络速率、延迟、进程、磁盘、IP 和运行时间。

## 当前实现

插件入口是 `src/index.ts`，通过 `TerminalDecorator` 挂载到 Tabby 终端标签页。

核心机制：

- 不使用 `sendInput()`，不会把采集命令输入到用户可见终端。
- 通过 Tabby SSH session 的 `openSessionChannel()` 和 `requestExec()` 打开独立 exec channel。
- 每次采集都是一次性命令，不在服务器常驻后台进程。
- 采集结果使用 tab 分隔文本返回，再由 `parseMetricPayload()` 解析。
- `mergeMetricPatch()` 合并局部数据，避免 fast/process/slow 任一采集刷新时清空其他区域。

刷新策略：

- `fast`：每 1 秒刷新 CPU、内存、交换、负载、网络字节计数、Tabby 到 SSH 服务器的 exec 往返延迟。
- `process`：每 3 秒刷新最多 30 条进程。
- `slow`：每 10 秒刷新 IP、运行时间、磁盘列表。

## 支持平台

Linux SSH 目标：

- CPU：`/proc/stat`，按相邻两次计数器差值计算；`iowait` 归入 idle，计数器强制输出十进制整数，避免长运行机器被 `awk` 格式化成科学计数法后 delta 变成 0。
- 负载：`/proc/loadavg`
- 内存/交换：`/proc/meminfo`
- 网络接口：`ip route get 1` 或 `/sys/class/net`
- 网络流量：`/sys/class/net/<iface>/statistics/*_bytes`
- 运行时间：`/proc/uptime`
- 磁盘：`df -P -h`
- 进程：`ps -eo ... --sort=-pcpu`

macOS/Darwin SSH 目标：

- CPU：`top -l 1`
- CPU 核数：`sysctl -n hw.ncpu`
- 内存：`vm_stat` 和 `sysctl -n hw.memsize`
- 网络接口：`route -n get default`
- 网络流量：`netstat -ibn`
- 本机 IP：`ipconfig getifaddr`
- 运行时间：`sysctl -n kern.boottime`
- 磁盘：`df -P -h`
- 进程：`ps axo ...`

不支持的场景：

- Tabby 本地默认终端窗口目前不会采集本机系统数据，因为它没有 SSH exec channel。

## IP 显示逻辑

公网 IP 和本机 IP 分开采集：

- `publicIp`：通过外部服务获取，按顺序尝试：
  - `https://api.ipify.org`
  - `https://ifconfig.me/ip`
  - `https://icanhazip.com`
  - `https://checkip.amazonaws.com`
- `localIp`：从目标机器的默认网卡或路由里获取。

展示优先级：

1. 公网 IP，标记为“公网”。
2. 原始 `ip` 字段。
3. 本机网卡 IP，标记为“本机”。

公网 IP 不能简单从网卡读取，因为很多云服务器网卡只有内网地址，公网地址由 NAT 或 EIP 映射。

## UI 行为

- 面板固定在左侧。
- 右边缘可拖拽调整宽度，范围是 260px 到 560px。
- 宽度保存在 `localStorage` 的 `tabby-status-width`。
- 面板和进程列表隐藏滚动条，但支持滚轮和触控板滚动。
- 紧凑文案统一单行显示，超出时省略。
- CPU 右侧显示百分比；内存、交换右侧同时显示百分比和容量，交换未启用时显示“未启用”。
- CPU、内存、交换行使用固定的“标签 / 进度条 / 辅助数值”三列布局，数值用更弱的亮度和等宽数字，避免抢占进度条视觉焦点。
- IP 行只显示 IP；采集错误显示在独立状态行，不覆盖已有 IP，正常状态下状态行为空。
- 进程列表使用固定“内存 / CPU / 命令”列宽，内存和 CPU 表头支持点击排序；进程名过长时只截断命令列；空进程刷新不会清空上一批进程数据。
- 磁盘列表的“可用/大小”表头和数值列右对齐，便于扫描容量。

## 安装

```bash
cd "/Users/endlessjy/Documents/AIGC/tabby-status"
npm install
npm run install-local
```

`install-local` 会执行：

1. `npm run build`
2. `npm pack`
3. 使用 `--legacy-peer-deps` 安装 tgz 到 `$HOME/Library/Application Support/tabby/plugins`，避免把 Tabby/Angular peer dependencies 复制进用户插件目录。

安装后重启 Tabby。
